#1
- git init: Initiate the project 
- git add <file_name> : Add the specific file to Staging area
- git add .: Add all files under current directory 
- git commit -m "message": Create a commit. 
   -m is a function of git commit to avoid extra step using text editor to define the commit
   "message": contains the description of the commit
